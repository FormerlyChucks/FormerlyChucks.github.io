=================
Local Development
=================

Clone the repo, change into the directory:

.. code-block:: bash

    git clone https://github.com/IThinkImOKAY/IThinkImOKAY.github.io && cd IThinkImOKAY.github.io

Run:

.. code-block:: bash

    python3 -m http.server

It should start running on http://localhost:8000
